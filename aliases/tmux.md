# tmux Terminal Aliases
## General
| Alias | Description |
|-------|-------------|
| `ts`  | Start Tmux. |
| `tq`  | Exit Tmux.  |
| `tsc` | Reload Tmux configuration. |
| `tlk` | List Tmux keys. |

## Sessions
| Alias | Description |
|-------|-------------|
| `tls` | List all sessions. |
| `tns` | Go back into session. |
| `tns` | Start new Tmux session with name. |
